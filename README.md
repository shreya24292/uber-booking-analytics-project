# uber-booking-analytics-project
NCR Ride Booking Data Analysis
Project Overview

This project analyzes 150,000 ride-booking records from the NCR region to identify customer behavior, operational efficiency, revenue trends, cancellation patterns, and service performance.

Objectives
Analyze booking trends
Understand cancellation behavior
Identify revenue drivers
Evaluate customer satisfaction
Discover peak demand periods
Generate business recommendations
